## Usage
**If you are new to repository/extensions, please read the [Getting Started guide](https://fam007e.github.io/extensions-source/docs/guides/getting-started) first.**

* You can add our repo by visiting the [Website](https://fam007e.github.io/extensions-source/add-repo).
* If you're not using any of the above, you can manually download and update extensions from the [listing page](https://fam007e.github.io/extensions-source/extensions/).
* Otherwise, copy & paste the following URL: https://raw.githubusercontent.com/fam007e/extensions-source/repo/index.min.json

## Requests

To request a new source or bug fix, [create an issue](https://github.com/fam007e/extensions-source/issues/new/choose).

Please note that creating an issue does not mean that the source will be added or fixed in a timely
fashion, because the work is volunteer-based. Some sources may also be impossible to do or prohibitively
difficult to maintain.

If you would like to see a request fulfilled and have the necessary skills to do so, consider contributing!
Issues are up-for-grabs for any developer if there is no assigned user already.
